Developer Survey Dashboard Project
Technology Trends Analysis

This project analyzes global developer survey data to identify current and future trends in programming languages, databases, cloud platforms, and web frameworks. The analysis includes data cleaning, visualization, dashboard creation, and interpretation of insights. The goal is to understand how the technology landscape evolves and which tools developers prefer today and expect to use in the future.

Project Structure
/data               → raw and cleaned datasets   
/images             → exported charts and dashboard screenshots    
README.md           → project documentation  


Project Objectives
Analyze developer preferences across multiple technology categories
Compare current usage vs. future interest
Visualize trends using dashboards and charts
Interpret insights and summarize key findings
Present results in a structured, professional format

Tools and Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
Some data cleaning and visualizations were completed inside Coursera Labs
Jupyter Notebook
Cognos Analytics
GitHub for version control and documentation

Key Insights
Core technologies remain stable across all dashboards: 
Python, JavaScript, SQL, PostgreSQL, AWS, React, and Node.js dominate both current usage and future preferences.
Emerging tools such as Rust, Go, Redis, and Supabase show growing interest but do not significantly change the overall rankings.
Cloud development is strongly centered around AWS, with Google Cloud and Azure following behind.
The JavaScript ecosystem continues to lead web development, supported by React, Node.js, and Next.js.
Demographics show that most respondents are young (25–34), hold a Bachelor’s degree, and are primarily located in Europe and North America.

Summary
This project provides a clear overview of the modern technology landscape and highlights stable long‑term trends in developer preferences. The results can support decision‑making for learning paths, hiring strategies, and technology adoption.
